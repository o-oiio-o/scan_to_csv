# Barcode Data Collection Tool

To facilitate barcode scanning data collection, this lightweight tool was developed.

## Features
A web-based barcode scanner/recorder that works instantly upon opening the webpage. Suitable for scenarios such as inventory counting and data collection.

## Highlights
- **Simple & Convenient**: No installation required, accessible directly via browser.
- **Continuous Scanning**: Supports sequential scanning without manual page refresh.
- **Auto-deduplication**: Automatically filters out duplicate entries.
- **Sound Feedback**: Provides sound prompts for successful scans.
- **CSV Export**: Generates data in CSV format for easy analysis and sharing.
